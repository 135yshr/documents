前回の記事では、猫語でプログラミングできる言語 **meow** の概要を紹介しました。今回は meow の基本文法を、FizzBuzz を段階的に作りながら解説します。

## meow をインストールする

Homebrew が使えるなら 1 行です。

```bash
brew install 135yshr/homebrew-tap/meow
```

Go がインストール済みなら、こちらでも OK です。

```bash
go install github.com/135yshr/meow@latest
```

インストール不要でブラウザから試すこともできます。

<https://135yshr.github.io/meow/playground/>

## 変数を宣言する（`nyan`）

meow では `nyan` キーワードで変数を宣言します。

```text
nyan message = "FizzBuzz"
nyan count = 15
nyan pi = 3.14
```

型の明示も可能です。

```text
nyan message string = "FizzBuzz"
nyan count int = 15
```

型を省略すると、右辺の値から推論されます。

meow には猫にちなんだリテラルもあります。

| meow       | 対応する値 | 由来             |
| ---------- | ---------- | ---------------- |
| `yarn`     | true       | 毛糸（猫の好物） |
| `hairball` | false      | 毛玉             |
| `catnap`   | nil        | 猫の昼寝         |

```text
nyan alive = yarn       # true
nyan sleepy = hairball   # false
nyan nothing = catnap    # nil
```

## 出力する（`nya`）

値を画面に出力するには `nya` を使います。Go の `fmt.Println` に相当します。

```text
nyan message = "FizzBuzz"
nya(message)
```

```text
FizzBuzz
```

数値もそのまま出力できます。

```text
nya(42)
nya(3.14)
```

```text
42
3.14
```

## 関数を定義する（`meow`）

関数は `meow` キーワードで定義します。引数には型を指定し、返り値がある場合は引数リストの後に戻り値の型を書きます。

値を返すには `bring` を使います。

```text
meow greet(who string) string {
  bring "Hello, " + who + "!"
}

nya(greet("Tama"))
```

```text
Hello, Tama!
```

`bring` は Go の `return` に相当するキーワードです。猫が獲物を持ってくるイメージです。

返り値がない関数は、戻り値の型を省略します。

```text
meow say_hello() {
  nya("Hello!")
}
```

## 条件で分岐する（`sniff` / `scratch`）

条件分岐には `sniff`（if）と `scratch`（else）を使います。

```text
nyan x = 10

sniff (x > 5) {
  nya("big")
}
```

`scratch` で else 分岐を書きます。

```text
sniff (1 > 2) {
  nya("greater")
} scratch {
  nya("not greater")
}
```

`scratch sniff` で else if を表現します。

```text
meow classify(x int) string {
  sniff (x > 20) {
    bring "big"
  } scratch sniff (x > 10) {
    bring "medium"
  } scratch {
    bring "small"
  }
}

nya(classify(15))
```

```text
medium
```

ネストも可能です。

```text
nyan x = 5

sniff (x > 0) {
  sniff (x < 10) {
    nya("single digit positive")
  } scratch {
    nya("large positive")
  }
} scratch {
  nya("non-positive")
}
```

## 繰り返す（`purr`）

ループには `purr` を使います。`..` でレンジを指定すると、その範囲を繰り返します。

```text
purr i (1..5) {
  nya(i)
}
```

```text
1
2
3
4
```

`1..5` は 1 以上 5 未満の範囲です。

## コメント

meow では `#` で行コメント、`-~ ~-` でブロックコメントを書けます。

```text
# これは行コメント

-~
  これはブロックコメント
  複数行にまたがって書ける
~-
```

コードの意図を残すときに使いましょう。

## FizzBuzz を完成させる

ここまでに学んだ文法を組み合わせて、FizzBuzz を段階的に作ってみましょう。

### Step 1: 変数と出力

復習を兼ねて、文字列を出力するところから始めます。

```text
nyan message = "FizzBuzz"
nya(message)
```

### Step 2: ループで 1 から 15 まで出力する

`purr` を使って数値を順番に出力します。

```text
purr i (1..16) {
  nya(i)
}
```

### Step 3: 条件分岐を追加する

`sniff` と `scratch` を使って FizzBuzz の判定ロジックを追加します。

```text
purr i (1..16) {
  sniff (i % 15 == 0) {
    nya("FizzBuzz")
  } scratch sniff (i % 3 == 0) {
    nya("Fizz")
  } scratch sniff (i % 5 == 0) {
    nya("Buzz")
  } scratch {
    nya(i)
  }
}
```

### Step 4: 関数に切り出す

ロジックを関数として切り出せば、再利用しやすくなります。範囲も広げてみましょう。

```text
meow fizzbuzz(n int) string {
  sniff (n % 15 == 0) {
    bring "FizzBuzz"
  } scratch sniff (n % 3 == 0) {
    bring "Fizz"
  } scratch sniff (n % 5 == 0) {
    bring "Buzz"
  } scratch {
    bring to_string(n)
  }
}

purr i (1..20) {
  nya(fizzbuzz(i))
}
```

`to_string` は数値を文字列へ変換する組み込み関数です。戻り値の型を `string` で統一するために使っています。

実行してみましょう。

```bash
meow run fizzbuzz.nyan
```

```text
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
```

## まとめ

この記事で紹介した meow の基本文法をまとめます。

| meow      | 対応する概念 | 使い方                      |
| --------- | ------------ | --------------------------- |
| `nyan`    | 変数宣言     | `nyan x = 42`               |
| `meow`    | 関数定義     | `meow f(x int) int { ... }` |
| `bring`   | return       | `bring value`               |
| `nya`     | print        | `nya(x)`                    |
| `sniff`   | if           | `sniff (条件) { ... }`      |
| `scratch` | else         | `scratch { ... }`           |
| `purr`    | for          | `purr i (1..10) { ... }`    |

FizzBuzz を通じて、変数宣言・関数定義・条件分岐・ループを一通り触れました。meow の文法は Go に似ていますが、すべてのキーワードが猫にちなんだ単語です。

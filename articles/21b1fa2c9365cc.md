猫語でプログラミングできる言語を作りました。名前は **meow** です。

```text
nyan greeting = "Hello, Nyantyu!"
nya(greeting)
```

これは変数 `greeting` に文字列を代入して出力するプログラムです。`nyan` が変数宣言、`nya` が出力。すべてのキーワードが猫語になっています。

ジョーク言語に見えますが、実際にコンパイルしてネイティブバイナリを生成できる、れっきとしたプログラミング言語です。

<https://github.com/135yshr/meow>

## なぜ作ったか

「Go の標準ライブラリだけでプログラミング言語を作れるのか」という疑問が出発点でした。

字句解析、構文解析、型検査、コード生成。言語処理系には多くの工程がありますが、Go の標準ライブラリには `iter.Seq` や `text/template` など、これらを支える道具が揃っています。外部依存ゼロで言語が作れるなら、やってみる価値があると思いました。

そしてどうせ作るなら、自分が好きなもの――猫をテーマにしよう、と。

## meow の仕組み

meow は `.nyan` ファイルを Go のソースコードにトランスパイルし、`go build` でネイティブバイナリを生成します。

```text
.nyan → 字句解析 → 構文解析 → 型検査 → コード生成 → .go → go build → バイナリ
```

Go にトランスパイルすることで、Go のランタイムとエコシステムをそのまま活用できます。生成されたバイナリはフルスピードで動作します。

## 30 秒で試す

### インストール

Homebrew が使えるなら 1 行です。

```bash
brew install 135yshr/homebrew-tap/meow
```

Go がインストール済みなら、こちらでも OK です。

```bash
go install github.com/135yshr/meow@latest
```

### Hello World を動かす

`hello.nyan` というファイルを作ります。

```text
nyan name = "Tama"

meow greet(who) {
  bring "Hello, " + who + "!"
}

nya(greet(name))
```

実行します。

```bash
meow run hello.nyan
```

```text
Hello, Tama!
```

ビルドしてバイナリを生成できます。

```bash
meow build hello.nyan -o hello
./hello
```

## 5 分でわかるコード例

### 条件分岐と関数

FizzBuzz を書いてみます。

```text
meow fizzbuzz(n int) string {
  sniff (n % 15 == 0) {
    bring "FizzBuzz"
  } scratch sniff (n % 3 == 0) {
    bring "Fizz"
  } scratch sniff (n % 5 == 0) {
    bring "Buzz"
  } scratch {
    bring to_string(n)
  }
}

purr i (1..20) {
  nya(fizzbuzz(i))
}
```

`sniff` で条件分岐、`scratch` で else。`purr i (1..20)` は 1 から 20 までのループです。

### リスト操作

関数型スタイルのリスト操作もサポートしています。

```text
nyan nums = [1, 2, 3, 4, 5]

nyan doubled = lick(nums, paw(x int) { x * 2 })
nya(doubled)  # => [2, 4, 6, 8, 10]

nyan evens = picky(nums, paw(x int) { x % 2 == 0 })
nya(evens)  # => [2, 4]

nyan sum = curl(nums, 0, paw(acc int, x int) { acc + x })
nya(sum)  # => 15
```

`lick` が map、`picky` が filter、`curl` が reduce。`paw` でラムダ式を書きます。

### 構造体とメソッド

```text
kitty Cat {
  name: string
  age: int
}

groom Cat {
  meow greet() string {
    bring self.name + " says meow!"
  }
}

nyan tama = Cat("Tama", 3)
nya(tama.greet())  # => Tama says meow!
```

`kitty` で構造体を定義し、`groom` でメソッドを追加します。

### エラーからの回復

```text
meow divide(a int, b int) int {
  sniff (b == 0) {
    hiss("division by zero")
  }
  bring a / b
}

nyan val = divide(10, 0) ~> 0
nya(val)  # => 0
```

`hiss` でエラーを投げ、`~>` でフォールバック値を指定します。ゼロ除算が起きても `0` が返ります。


`meow transpile` を使えば、生成される Go のコードを確認できます。言語がどう変換されるのか覗いてみるのも面白いはずです。
